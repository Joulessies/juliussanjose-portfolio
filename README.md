# Julius San Jose Portfolio

[https://www.juliussanjose.tech/](https://www.juliussanjose.tech/)

## Tech Stack

- [Next.js](https://nextjs.org/) - Main framework for building and structuring the portfolio site.
- [GSAP](https://gsap.com/)– Handles advanced page transitions for smooth navigation.
- [motion](motion.dev) – Manages animations for interactive and dynamic UI elements.
- [Lucide Icons](https://lucide.dev/) – For consistent and modern iconography.
